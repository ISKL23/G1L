Overview

The mod manager built for Gothic 1 Remake

Lightweight, fast and fully portable. Drop a mod archive or folder onto the window — G1L detects what kind of mod it is and installs it exactly where the game expects it. No more digging through game folders by hand. Most managers only handle UE4SS or paks; G1L installs and manages every Gothic 1 Remake mod type — correctly.

Workflow

Install & manage

  • ➤Drag-and-drop, or pick files. Drop archives or a mod folder, or click + to choose several. G1L detects the type and puts it in the right folder.
  • ➤UE4SS done for you. Installs the correct experimental build for Gothic 1 Remake (regular zip, never zDEV), respects an existing nested install, and can fix a broken one.
  • ➤Pak toggling. Disable & re-enable pak mods safely — G1L renames the full .pak/.ucas/.utoc/.sig set; nothing is ever deleted.
  • ➤Loose game files are managed. Images, Story and Textures replacements are installed conservatively: originals are backed up, disabling restores them, enabling restores the mod payload.
  • ➤Engine.ini tweaks done right. Merged into your real Engine.ini as removable marked blocks. Disable the mod and only its block is removed — your other settings stay.
  • ➤Built-in config editor. Edit mod settings in the launcher — even values written inside a UE4SS main.lua, rewriting only the value in place so the script never breaks. A raw text tab is always there too.
  • ➤In-game console (opt-in). One toggle in Settings turns on the Unreal console and cheat commands via the UE4SS console built-ins. Off by default — a developer / cheat feature, separate from the UE4SS debug log. Requires UE4SS.

Getting started

How to use

1

Run ISKLLauncher.App.exe — the game is detected automatically.

2

If prompted, install UE4SS from Settings. Lua/DLL mods need it; pak / movie / tweak / preset mods do not.

3

Drag a mod archive, .pak, or folder onto Available mods — or click + to pick files.

4

Enable the mods you want and set the order if needed.

5

Click the config icon if a mod has editable settings.

6

Hit PLAY. Done.

Help

FAQ

Updates not showing?
  • Update checks need a free Nexus API key — paste it in Settings → Nexus Updates (Nexus: Account → Site preferences → API Keys).
  • The mod also needs a Nexus ID. G1L reads it from the original download name; if missing, set it via right-click → Edit Nexus metadata.
  • G1L reads the mod's latest file version, so updates show even when the author didn't bump the page version.
Mods not loading?
  • Open Settings → Install / Update UE4SS for Lua/DLL mods, or hit Fix UE4SS if it was installed into the wrong folder.
  • Pak, movie, Engine.ini and ReShade preset mods do not need UE4SS.
  • BP mods need SimpleModLoader; fyryNy plugins need fyryNy's Loader.
  • Watch the yellow banner — G1L checks your setup and tells you what's missing or broken.
Is it safe? Why does my antivirus flag it?
G1L ships as a self-contained single-file .NET app. Some antivirus engines flag any single-file executable as “packed” — a known false positive for this format. The launcher only touches your Gothic 1 Remake mod folders. If Nexus temporarily reviews a new file, use the official site or GitHub mirror and scan the build yourself on VirusTotal.
Launcher won't open / mods won't drag in?
Read the Troubleshooting & FAQ first — it covers 95% of issues. Still stuck? Use About → Copy diagnostics and paste it into a comment.
