# G1L official website

This repository contains the public static website for **G1L — ISKL Gothic 1 Remake Launcher**.

Live site: <https://iskl23.github.io/G1L/>

## What this repository is

- Official website for G1L.
- Download mirror page pointing to the latest official release.
- Static HTML/CSS/JS only.
- No launcher source code is stored here.

## Downloads

Use the latest official build from:

- Website: <https://iskl23.github.io/G1L/>
- GitHub releases: <https://github.com/ISKL23/G1L-Releases/releases/latest>
- Nexus Mods: <https://www.nexusmods.com/gothic1remake/mods/37>

If Nexus temporarily quarantines a new upload while its scanner reviews the file,
use the website or GitHub release mirror. You can also verify any downloaded build
yourself on VirusTotal.

## Using parts of this website

You may use this website source as reference or adapt small parts of the layout for
your own project.

Please credit **ISKL** and link back to the project if you borrow visible design,
copy, structure, or code from this site.

This permission applies only to the website files in this repository. It does not
make the G1L launcher source code open source.

## Launcher source code

The G1L application itself is a separate proprietary tool. Its source repository is
private. Do not decompile, copy, redistribute, or modify the launcher outside the
official release terms.
